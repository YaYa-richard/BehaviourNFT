# BehaviourNFT

Key words: Hardhat, React, MongoDB, IPFS, Hardhat testnet

```shell
cd backend && yarn install && cd ../frontend && yarn install && cd ../contract && yarn install
```

## backend

## frontend


```shell
yarn hardhat node
```

Deployment

```shell
yarn hardhat run ignition/modules/deploy.js --network localhost
```
